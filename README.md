
# Office-Leave

An online employee leave management in ASP.Net Core MVC



## Run Locally

Clone the project

```bash
  git clone https://github.com/MoQuayson/Office-Leave.git
```

Go to the project directory

```bash
  cd Office-Leave
```

Install dependencies

```bash
  libman install
```

Start the server

```bash
  dotnet run
```
